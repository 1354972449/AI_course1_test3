# 安徽宣城公共资源交易平台运营商项目监测工具

定时抓取宣城市公共资源交易中心网站上与运营商相关的政府采购项目，筛选需求公示和成交公告，生成Excel报告。

## 功能特性

- ✅ 浏览器自动化方式抓取数据
- ✅ 筛选需求公示（采购意向公开）项目
- ✅ 筛选招标公告项目
- ✅ 筛选成交公告（中标结果）项目
- ✅ 智能识别运营商可参与项目
- ✅ 定时任务：每周五自动执行
- ✅ Excel格式输出报告

## 项目结构

```
├── scraper.py                      # 数据抓取脚本
├── weekly_monitor.py               # 定时监测脚本
├── 宣城运营商项目清单.py            # Excel生成脚本
└── README.md                       # 项目说明
```

## 使用方法

### 1. 手动运行监测任务

```bash
python weekly_monitor.py
```

### 2. 生成项目清单Excel

```bash
python 宣城运营商项目清单.py
```

### 3. 定时任务（每周五17:00自动执行）

任务已配置为每周五自动执行，会抓取最新数据并生成Excel报告。

## 数据来源

- **网站**：宣城市公共资源交易中心
- **网址**：https://ggzyjy.xuancheng.gov.cn
- **分类**：政府采购 > 招标公告/采购意向公开/中标(成交)公告

## 输出报告

Excel文件包含以下字段：
- 序号
- 项目名称
- 信息类型（招标公告/采购意向公开/成交公告）
- 发布时间
- 投标截止日期
- 项目概述
- 运营商参与方向

## 依赖环境

- Python 3.x
- openpyxl（用于生成Excel）
- 浏览器（用于自动化抓取）
